# emozi

emozi is a simple command line tool to generate emojipasta (copypasta but a lot of emojis).

Inspired by the [EmojiPastaBot](https://github.com/Kevinpgalligan/EmojipastaBot) on reddit.

Example:
```
cat example.txt | emozi
```
